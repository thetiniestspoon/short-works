# Short Works

A curated collection of short literary works by Shawn Jordan.

Prose poems, flash fiction, poetry, and hybrid pieces — arranged in an emotional arc from warmth through darkness back to wonder.

**Read online:** [thetiniestspoon.github.io/short-works](https://thetiniestspoon.github.io/short-works/)

## Building

```bash
python build-reader.py
```

This reads `content/*.md` and generates `reader/index.html`. Open it in a browser to preview.

## Deploying

Push to `main` — GitHub Actions deploys `reader/` to GitHub Pages automatically.
